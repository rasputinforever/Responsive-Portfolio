Portfolio

Castaway: Flight of the Phoenix

This team project is a "Choose your own Adventure" game which is an MVC-organized application. An express (npm) server serves express-handlebars based pages which pulls data from a MySQL database using sequelize. Clever CSS and express-handlebars trickery allows for a smooth, fun user experience.

Employee Manager

Employee Manager is a Command Line application which uses Node.js to step through and deliver MySQL tables which summarize an organization. Employees, Departments, Salaries, Managers: everything you need to track your staff can be built here. Check it out!

Auto-Team-HTML-Builder

This application is a command-line application that uses Node.js to step-wise walk a user through building a small team's website. By asking for each employee's information the application is able to output an HMTL page with all styling and formatting included!

Auto ReadMe

This application is, once again, a node.js based command line application that step-wise helps a user build a standardized README.MD file, useful for when putting together a new repository. Try it out!

My Weather Dashboard

This project utilizes asynchronous page loading that delivers various weather information for a given city. It heavily relies on JQuery in its script, not to mention web API, server side APIs, and pure.css page style cdn.

Work Day Planner

This project utilizes a server-side API to deliver time-dependent page styling. JQuery was in use to add in minor "animated" buttons, plus Bootstrap for basic page styling.

Entertainment Guide

This group project puts together the minds of four individuals in creating a streamlined entertainment guide. Utilizing Bulma.io and server-side APIs, the group delivered a clean looking page by individually contributing JavaScript, css, HTML, and general workflow organization.
